# Pynosaur Documentation

Modern CLI tools with better UX than traditional Unix utilities.

[Back to Main](../)

## Tools

- [pget](https://github.com/pynosaur/pget) - Package manager
- [see](https://github.com/pynosaur/see) - File viewer
- [purl](https://github.com/pynosaur/purl) - URL transfer
- [yday](https://github.com/pynosaur/yday) - Day of year
- [doc](https://github.com/pynosaur/doc) - Documentation viewer

## Get Started

```bash
pget install see
see README.md
```
